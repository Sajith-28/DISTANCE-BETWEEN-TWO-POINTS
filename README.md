## AIM:

## ALGORITHM:

### Step 2:

### Step 3: 
Substitute the values in the distance formula  ![formula](/formula.JPG)
### Step 4: 
Print the distance.
### Step 5: 
End the program
### PROGRAM:
```PYTHON
DEVELOPED BY : SAJITH AHAMED F

import math
l1=[4,2]
l2=[10,6]
d=math.sqrt(((l2[0]-l1[0])**2)+((l2[1]-l1[1])**2))
print(f"{d:.2f}")
```
